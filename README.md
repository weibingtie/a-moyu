# LS摸鱼神器 - 上班摸鱼专用炒股盯盘小工具

一个专为「上班摸鱼」场景设计的轻量级股票桌面小挂件：  
贴在屏幕边角，不挡窗口、不抢焦点，静静帮你盯盘。

## ✨ 特色功能

- **桌面悬浮挂件**  
  - 无边框小窗口，可拖拽到任意位置（推荐右上角 / 侧边）  
  - 支持置顶显示，不影响正常办公软件  

- **实时盯盘展示**  
  - 支持沪深京多市场代码（`sh600000` / `sz000001` / 各类指数等）  
  - 显示：股票代码、股票名称、现价、涨跌幅、量比等核心信息  
  - 红涨绿跌，重要数值自动高亮  

- **摸鱼友好设计**  
  - 启动后窗口极简，不占地方、一眼看完  
  - 支持双击快速隐藏/显示  
  - 右键菜单：一键设置 / 刷新 / 退出  
  - 提供全局快捷键（可配置），不用切回窗口也能弹出/隐藏  

- **高度可定制**  
  - 自选股列表管理（添加、删除、上下移动）  
  - 自由选择显示字段（代码、名称、涨跌幅、量比等）  
  - 字体、字号、行距可调，适配不同显示器和视力  
  - 背景色 & 透明度可调，可伪装成桌面装饰  

- **轻量、独立**  
  - 纯本地小工具（通过公开行情接口获取数据）  
  - 一个 exe 即可运行

## 🚀 适用场景

- 上班需要「假装在看报表」，其实在看行情  
- 家里多屏幕，想在角落挂一个实时盘口小条  
- 不想开复杂的交易软件，只想随手看几只股票 / 指数  
- 想做自己的轻量盯盘桌面工具 

> 提示：部分杀毒软件可能误报，需要手动添加信任。

## 🧩 快速上手

- **添加股票**  
  右键窗口 → `设置` → `自选列表`  
  支持输入 `600519` / `sh600519` 等格式，程序会自动规范化。

- **调整显示字段**  
  右键窗口 → `设置` → `显示数据`  
  勾选你关心的指标（现价、涨跌幅、量比等）。

- **调整外观**  
  右键窗口 → `设置` → `外观`  
  可设置：是否显示表头、字体/字号、行距、颜色、透明度等。

- **快捷键（可选）**  
  `设置` → `快捷键`：配置显示/隐藏、打开设置的全局快捷键  
  例如：`Ctrl+Alt+S` 切换显示/隐藏，`Ctrl+Alt+O` 打开设置。
